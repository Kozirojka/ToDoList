Завдання:
Розробіть простий RESTful Web API за допомогою ASP.NET Core, який дозволятиме керувати списком задач (ToDo).

Функціональні вимоги:
 • Модель задачі:
Створіть модель Task (або ToDo) з наступними властивостями:
 • Id – унікальний ідентифікатор (тип Guid або int)
 • Title – назва задачі
 • Description – опис задачі
 • IsCompleted – статус виконання (тип bool)
 • DueDate – опціональна дата виконання
 • API-методи:
Реалізуйте наступні HTTP-ендпоїнти:
 • GET /api/tasks – отримання списку всіх задач
 • GET /api/tasks/{id} – отримання задачі за ідентифікатором
 • POST /api/tasks – створення нової задачі
 • PUT /api/tasks/{id} – оновлення існуючої задачі
 • DELETE /api/tasks/{id} – видалення задачі за ідентифікатором
 • Зберігання даних:
Для зберігання даних використовуйте in-memory колекцію (не інтегруйте базу даних).

Нефункціональні вимоги:
 • Дотримання чистої архітектури та принципів SOLID.
 • Чіткий, структурований код з відповідними коментарями.
 • Надання інструкцій для запуску проекту (README файл).

Бонус (необов’язково):
 • Реалізуйте валідацію вхідних даних для POST та PUT запитів.
 • Покрийте основні функції юніт тестами.

Термін виконання:
Надішліть, будь ласка, виконане завдання протягом 5 робочих днів.
